Blinkit Sales Analysis Dashboard (Excel)
Project Overview

This project is an interactive Excel dashboard developed to analyze Blinkitâ€™s sales performance across multiple dimensions including item type, fat content, outlet size, outlet location, and outlet type.
The dashboard consolidates raw sales data into KPIs and visual insights to support business and operational decision-making.

The focus of this project is analytical accuracy, interactivity, and business insight, built entirely using Microsoft Excel.

Business Problem

Blinkit operates through different outlet formats and locations while offering a wide range of products.
Stakeholders need answers to key questions such as:

Which product categories generate the highest sales?

How do outlet size and location impact revenue?

What is the sales contribution of low-fat vs regular products?

How has outlet performance evolved over time?

Which outlet types deliver better average sales and volume?

This dashboard addresses these questions in a single, interactive view.

Dataset Description

The dataset contains structured sales records with attributes such as:

Item Type

Fat Content (Low Fat / Regular)

Outlet Size (Small / Medium / High)

Outlet Location (Tier 1 / Tier 2 / Tier 3)

Outlet Type (Supermarket Type 1/2/3, Grocery Store)

Sales Value

Item Count

Customer Rating

Data Preparation

All data processing was performed in Microsoft Excel, including:

Cleaning missing and inconsistent values

Standardizing item and outlet categories

Creating calculated fields for KPIs

Building Pivot Tables as the analytical foundation

Connecting slicers to all relevant visuals

Key Performance Indicators (KPIs)

These KPIs provide an instant overview of overall business performance.
